# Curso TMW Git & GitHUb 2025

Um para curso iniciantes aprenderem a trabalhar com versionamento de código e repsositórios remotos com GitHub.

Além disso, vamos trabalhar com GitFlow ao final do curso e Visual Studio Code.

Confira tudo o que temos no nosso YouTube. É grátis!

## Fluxo de trabalho git local

01. git checkout -b
02. cria ou atualiza arquivos
03. status do git
04. git adicionar arquivos
05. status do git
06. git commit -m "minha mensagem"
07. git checkout main
08. git merge nova_branch

## Fluxo de trabalho GitHub <> Local (projeto próprio ou da sua empresa)
01. git clone
02. git checkout -b <nova_branch>
03. alterações de arquivos
04. status do git
05. git adicionar arquivos
06. status do git
07. git commit -m "nova mensagem"
08. git push origin <nova_branch>
09. abrir pull request no GitHub para principal
10. origem <nova_branch>
11. git checkout main
12. git branch -D <nova_branch>

## Fluxo de trabalho GitHub <> Local (projetos open-source)
01. Fork do projeto para seu próprio github
02. git clone
03. git checkout -b <nova_branch>
04. alterações de arquivos
05. status do git
06. git adicionar arquivos
07. status do git
08. git commit -m "nova mensagem"
09. git push origin <nova_branch>
10. abrir Pull request no GitHub do branch fork para o principal do projeto original
11. origem <nova_branch>
12. git checkout main
13. git branch -D <nova_branch>
